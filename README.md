# Better India

Reddit-inspired community UI rebuilt with React and Tailwind CSS.

## To install all the Dependencies 
- `npm install`

## Scripts

- `npm run dev`
- `npm run build`
- `npm run lint`


## Stack

- React
- Tailwind CSS
- Parcel

## UI Direction

- subreddit-style hero banner and top bar
- feed cards with vote rails and engagement actions
- right rail for rules, mods, and trending topics
- Tailwind utility styling instead of the previous Vite-era CSS
